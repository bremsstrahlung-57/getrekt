# GETrekt

A simple, lightweight HTTP server written in C.

---

## Features

- **Serves static files:** HTML, CSS, JS
- **SQLite-backed data storage**
- **Full CRUD support:** GET, POST, PUT, DELETE
- **JSON responses**
- **Static routing**
